HTML

  • HTML creates content structure for a webpage.
  • The head element contains information about the webpage.
  • The head contains info that helps the browser render the page correctly. Not visible to users.
  • Browsers use metadata to effectively serve the webpage to users.
  • The body element represents the visible content shown to the user.
  • It's important to use headers in number order for accessibility.
  • HTML attributes define additional info about elements. They are defined in the beginning element, never the closing
